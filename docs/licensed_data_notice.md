# Licensed-data notice

This artifact does not redistribute raw OptionMetrics IvyDB US option-chain data.

The empirical study uses licensed option-chain data to construct date-level standardized option-book losses and derived diagnostics. Researchers with licensed access to the same data can reproduce the study by applying the documented filters, book-construction rules, next-day marking hierarchy, loss normalization, forecasting protocol, and configuration files.

This GitHub-light artifact keeps aggregate review outputs and generated figures. Large date-level derived panels are omitted. Before any public release, check the data license and consider publishing only code, aggregate tables, figure inputs, synthetic data, and final figures if redistribution of date-level derived panels is restricted.

Do not upload raw option-chain files, contract-level quotes, personal data paths, or credential-bearing configuration files to a blind-review repository.
