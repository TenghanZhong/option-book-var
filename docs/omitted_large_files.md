# Omitted large files

Large date-level outputs are not included because they are unnecessary for table-level verification and substantially increase the repository size. They can be regenerated by running `src/run_option_book_var.py` with licensed OptionMetrics-compatible raw data.

|---|---:|---|
| `ablation_results/book_var_calibration_memory_ablation_detail.csv` | 49.56 | detailed date-level ablation panel |
| `results/qqq/book_var_results_v25_paper_ready.csv` | 24.93 | date-level result panel |
| `results/spx/book_var_results_v25_paper_ready.csv` | 19.62 | date-level result panel |

| `results/spx/book_var_marking_intersection_dates_v25.csv` | 2.22 | date-level marking-intersection panel |

The repository retains the smaller aggregate outputs needed to verify summary, baseline, backtest, book-quality, yearly, crisis, feature-audit, calibration-memory, and marking-summary results. The omitted files contain derived results rather than redistributable raw OptionMetrics records.
